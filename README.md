# lame-android-build

    the script to build lame lib for android platform.

# usage

* copy lame project to the same dir as build-lame.sh, like:

```  
    
    |---------------
    |--lame
    |
    |--build-lame.sh
   
``` 

* open the build-lame.sh and set your NDK_ROOT path

* build
```shell

    ./build-lame.sh

```

# configuration

  * neon support, set your CFLAGS
